## **Хранение данных**

В этом разделе выбирается тип локального хранилища.

---

## **SQLite**

По умолчанию используется файл базы данных SQLite.

Преимущества:

* не требует сервера;

* моментальное чтение/запись;

* легко переносится.

Файл расположен обычно в:

`%AppData%\OrdersCreator\`

Рекомендуется периодически делать резервную копию.
