## **Горячие клавиши**

| Действие | Клавиша |
| ----- | ----- |
| Создать отчёт | **F12** |
| Отмена / очистка заказа | **DEL** |
| Открыть справку | **F1** |
| Добавить новый товар (красный режим) | **Enter** |
| Удалить строку в таблице | кнопка **X** |
| Сменить контрагента | Alt \+ C (если добавишь) |
