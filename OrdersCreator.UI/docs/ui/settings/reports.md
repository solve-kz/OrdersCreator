## **Настройки раздела «Отчёты по заказам»**

Этот раздел отвечает за структуру и сохранение Excel-отчётов.

---

### **📁 Корневая папка для отчётов**

Указывает папку, куда будут сохраняться все сформированные файлы.  
 Можно выбрать через кнопку **«Выбрать»**.

---

### **📆 Создавать подпапку с датой**

Если включено, каждый день создаётся отдельная папка:

`<КорневаяПапка>\2025-01-18\`

Это упрощает навигацию и архивацию.

---

### **📄 Путь к .xlsx-шаблону**

Шаблон отчёта, содержащий структуру документа:

* логотип,

* шапку заказа,

* столбцы таблицы,

* формулы.

Программа заполняет в нём:

* дату,

* контрагента,

* таблицу позиций,

* итоги веса.

---

### **🏷 Маска имени файла**

Позволяет формировать понятные имена отчётов.

Примеры переменных:

| Переменная | Значение |
| ----- | ----- |
| `{CustomerName}` | имя выбранного контрагента |
| `{Date:yyyy-MM-dd}` | дата |
| `{Time:HHmm}` | время |
| `{Count}` | количество строк заказа |

Пример маски:

`Отчёт_{CustomerName}_{Date:yyyy-MM-dd}.xlsx`

---

### **📂 Открывать отчёт после сохранения**

Если включено — Excel запускается автоматически.
