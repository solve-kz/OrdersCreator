# Быстрый старт

Этот раздел предназначен для быстрого освоения программы  
**«Складской терминал»** и описывает полный рабочий цикл оператора — от запуска до формирования отчёта.

Если вам нужно подробное описание элементов интерфейса, см.  
[Главное окно и элементы управления](ui/main-window.md).

---

## Назначение программы

Программа предназначена для:
- приёма и учёта продукции по штрих-кодам;
- автоматического извлечения веса;
- формирования заказов по контрагентам;
- получения итогового отчёта.

Основной принцип работы — **сканирование штрих-кодов без ручного ввода веса**.

---

## Что нужно перед началом работы

Перед первым использованием убедитесь, что:
- заполнены справочники товаров и категорий;
- добавлены контрагенты;
- подключён сканер штрих-кодов.

!!! note "Примечание"
    Если товар отсутствует в базе, его можно добавить прямо во время работы  
    (см. [Добавление нового товара](ui/main-window.md/#-добавление-нового-товара)).

---

## Запуск программы

После запуска открывается **главное окно программы**.

![Главное окно программы](images/main_form.png)

Интерфейс разделён на:
- верхнюю панель выбора контрагента;
- информационную панель сканирования;
- таблицу текущего заказа;
- итоговую панель.

Подробное описание см.  
[Главное окно и элементы управления](ui/main-window.md/#общий-вид-главного-окна).

---

## Шаг 1. Выбор контрагента

Перед началом сканирования необходимо выбрать контрагента.

1. Найдите поле **Контрагент** в верхней части окна.
2. Выберите нужного контрагента из списка.

!!! warning "Важно"
    Пока контрагент не выбран, сканирование заблокировано.  
    Программа выведет сообщение **«Выберите контрагента!»**.

См. подробнее:  
[Верхняя панель и выбор контрагента](ui/main-window.md#верхняя-панель-и-выбор-контрагента)

---

## Шаг 2. Сканирование товара

После выбора контрагента программа переходит в состояние:

**«Готов к сканированию!»**

![Готов к сканированию](images/panel3.png)

1. Отсканируйте штрих-код товара.
2. Программа автоматически:
   - определит товар;
   - извлечёт вес;
   - добавит позицию в заказ.

Данные текущего товара отображаются в информационной панели.

См. подробнее:  
[Информационная панель сканирования](ui/main-window.md#информационная-панель-сканирования)

---

## Шаг 3. Контроль позиций заказа

Все отсканированные товары отображаются в таблице заказа.

В таблице можно:
- просматривать список позиций;
- контролировать вес каждой строки;
- удалять ошибочные позиции.

!!! note "Примечание"
    Удаление строки влияет только на текущий заказ и не удаляет товар из базы.

См. подробнее:  
[Таблица позиций заказа](ui/main-window.md#таблица-позиций-заказа)

---

## Шаг 4. Добавление нового товара (при необходимости)

Если отсканированный штрих-код отсутствует в базе, программа откроет окно добавления товара.

![Товар не найден](images/addnewproduct.png)

1. Введите наименование товара.
2. Выберите категорию.
3. Нажмите **ДОБАВИТЬ ТОВАР (ENTER)**.

!!! success "Готово"
    Новый товар будет сохранён и автоматически добавлен в текущий заказ.

См. подробнее:  
[Добавление нового товара](ui/main-window.md#добавление-нового-товара)

---

## Шаг 5. Завершение работы и отчёт

После завершения сканирования:

1. Проверьте итоговые значения в нижней части окна:
   - количество позиций;
   - количество мест;
   - общий вес.
2. Нажмите **СОЗДАТЬ ОТЧЁТ (F12)**.

![Итоги заказа](images/main_form.png)

Для отмены текущего заказа используйте **ОТМЕНА (DEL)**.

См. подробнее:  
[Итоговая панель заказа](ui/main-window.md#итоговая-панель-заказа)

---

## Типовой рабочий цикл (кратко)

1. Запустить программу  
2. Выбрать контрагента  
3. Сканировать товары  
4. При необходимости добавить новый товар  
5. Проверить итоги  
6. Создать отчёт
