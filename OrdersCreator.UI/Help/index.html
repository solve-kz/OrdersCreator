    Добро пожаловать в Складской Терминал

    Складской Терминал — это настольное приложение для операторов и кладовщиков, которое упрощает процесс учёта грузов по заказам. Оно позволяет сканировать штрих‑коды товаров, автоматически извлекать вес из кода упаковки, суммировать количество мест и общий вес, а также формировать отчёт в формате Excel по заранее настроенному шаблону.

    Программа ориентирована на работу на складе или в экспедиции, где важно быстро и точно регистрировать отгружаемые товары и сохранять удобный для клиентов отчёт.

    Основные возможности

    • Сканирование штрих‑кодов: при помощи обычного USB‑сканера оператор считывает код товара, программа автоматически вычисляет вес и добавляет позицию в список заказа.

    • Учёт веса и количества: приложение считает количество мест и суммарный вес как по каждому отдельному товару, так и по заказу в целом.

    • Работа с неизвестными товарами: если код товара отсутствует в справочнике, приложение позволит быстро добавить новый товар, указав наименование и категорию, и продолжить работу.

    • Создание отчёта: по завершении сканирования оператор может создать отчёт в формате .xlsx на основе собственного шаблона; отчёт сохраняется в указанную папку и может автоматически открываться после сохранения.

    • Справочники: приложение содержит простые редакторы для категорий товаров, товаров со штрих‑кодами и контрагентов. Это упрощает подготовку базы данных и поддержку её в актуальном состоянии.

    • Гибкие настройки: в меню «Настройка» можно указать путь к шаблону отчёта, маску имени файла, поведение при неизвестном товаре, таймаут ввода с сканера, тип хранилища данных и другие параметры.

    Структура справки

    Справочная система разбита на отдельные разделы, которые можно изучать в любом порядке:

    • Начало работы — установка программы, подключение сканера и первый запуск.

    • Обзор интерфейса — описание главного окна, панелей и кнопок.

    • Работа с заказом — пошаговое руководство по сканированию, учёту веса и формированию отчёта.

    • Справочники — управление категориями товаров, товарами и контрагентами.

    • Настройки — конфигурация отчётов, поведения и хранения данных.

    • Горячие клавиши — список основных сочетаний клавиш, ускоряющих работу.

    • Типичные сценарии — примеры реальных ситуаций на складе и как с ними справляться.

    • Диагностика и устранение неполадок — советы по решению часто возникающих проблем.
